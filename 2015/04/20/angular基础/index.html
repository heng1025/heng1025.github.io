angular基础入门

  1. 背景
    • angularJS 是Misko Hevery 等人创建，在2009年时被谷歌收购，由一个全职团队维护；
    • angularJS 可以轻松构建SPA 页面，通过指令扩展了HTML，可以实现数据的双向绑定；
    • MVC MVVM 模块化 指定系统 数据双向绑定；

  2. 起步

    • 下载 Angular.js 的包 npm install angular 或 bower install angular，然后在页面中进行引用。
    • 永远的Hello World,在输入框中输入 Hello world会显示出来，这里也实现了双向绑定

  3. 简单说明

    • ng-app是一个入口指令，用来管理页面中的元素
    • ng-model={name} 是把文本框的值绑定在name上，当文本框的值发生改变时，而 也会相应的改变，即双向绑定；
