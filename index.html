angular基础

angular基础入门

  1. 背景
    • angularJS 是Misko Hevery 等人创建，在2009年时被谷歌收购，由一个全职团队维护；
    • angularJS 可以轻松构建SPA 页面，通过指令扩展了HTML，可以实现数据的双向绑定；
    • MVC MVVM 模块化 指定系统 数据双向绑定；

  2. 起步

    • 下载 Angular.js 的包 npm install angular 或 bower install angular，然后在页面中进行引用。
    • 永远的Hello World,在输入框中输入 Hello world会显示出来，这里也实现了双向绑定

  3. 简单说明

    • ng-app是一个入口指令，用来管理页面中的元素
    • ng-model={name} 是把文本框的值绑定在name上，当文本框的值发生改变时，而 也会相应的改变，即双向绑定；

node基础

node.js的基础知识

  1. node.js的简介
    Node.js是一个基于Chrome JavaScript运行时建立的平台，用于方便地搭建响应速度快、易于扩展的网络应用。
    Node.js 使用事件驱动， 非阻塞I/O 模型而得以轻量和高效，非常适合在分布式设备上运行的数据密集型的实时应用。
  2. node.js的版本说明
    node.js目前有两个版本 一个是长期支持版,目前最新版本是的4.4.2；另一个是稳定版，最新版本是5.10.0。
  3. node.jsd开发环境配置
  • 在进行node开发之前必须配置node的开发环境，首先在node.js的官网上下载node.js的安装包，下载地址https://nodejs.org/en/.安装后，node.js会默认添加到环境变量。安装完成之后在控制台下输入node -v会得到以下结果。
    v4.4.0
  • 当我们安装完后node后，node自带一个包管理工具npm,node package manager(node.js的包管理工具)。npm的官网https://www.npmjs.com/.npm是全球最大的开源库系统。
  • 由于node.js的版本更新比较快，变化也比较大，node.js的官方也考虑到了这一点，故当我们安装好了之后，我们可以在npm的官网上下载一个模块nvm,它是专门用来管理node.js的版本。
    • nvm的使用说明：所有的操作均在操作系统的控制台下进行。

总结

  • 在正式学习node之前必须配置好node的环境，以致于在后续的学习中能够专注于自己的代码。
  • 在学习之前务必搞清楚node ，npm, nvm 是什么，怎么用，最有效的方式就是查看它们的官网。
